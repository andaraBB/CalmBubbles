# Calm Bubbles
NeoForge 1.21.1 mod disabling the push/pull effects of vanilla bubble columns.

- **Minecraft**: 1.21.1
- **Loader**: NeoForge 21.1.x
- **License**: Apache-2.0

## Install
Drop the JAR in your `mods/` folder (client or server). No config required.
